# AutoHotkey scripts
  
This repository contains AutoHotkey scripts.  
  
## Shortcuts  
  
- \[CapsLock\] - change input language;  
- \[Insert\] - launches cmd;  
- \[ScrollLock\] - launches git-bash.  
  
## Dependencies  
  
- [awk](http://gnuwin32.sourceforge.net/packages/gawk.htm)
